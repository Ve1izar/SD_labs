# Локальний Трекер Звичок та Завдань

## Опис
Вебзастосунок на базі Streamlit для організації часу, відстеження регулярних звичок та разових завдань. Проєкт розроблено в рамках лабораторних робіт з ООП.

## Функціональність
- Додавання, редагування та видалення завдань/звичок (CRUD)
- Динамічне відстеження частоти (щоденно, щотижнево, щомісячно)
- Сповіщення про виконання (патерн Observer)
- Збереження даних у локальну БД SQLite (патерн Singleton)
- Перегляд статистики та історії логів

## Запуск проєкту
```bash
pip install streamlit
streamlit run main.py
```

## Тестування
```bash
python -m unittest discover tests/
```
